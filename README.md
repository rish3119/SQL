# SQL Practice Repository

This repository contains a collection of SQL scripts for learning, practicing, and mastering database queries.  
It is designed to help beginners and intermediate learners strengthen their SQL foundations.

## 📂 Structure
- `queries/` → Contains practice SQL queries (SELECT, JOIN, GROUP BY, etc.)
- `schemas/` → Database schema definitions for sample tables
- `examples/` → Real-world query examples and solutions

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sql-practice.git


Navigate into the repo:

cd sql-practice

Run SQL scripts using your preferred database system:

MySQL: mysql -u username -p < queries/example.sql

PostgreSQL: psql -U username -d dbname -f queries/example.sql

Oracle SQLPlus*: @queries/example.sql

📘 Topics Covered
Basic SELECT

Filtering with WHERE

Sorting with ORDER BY

Aggregations (SUM, COUNT, AVG)

Joins (INNER, LEFT, RIGHT)

Subqueries

Schema creation

🛠 Requirements
Any SQL-compatible database (MySQL, PostgreSQL, Oracle, SQLite)

Basic knowledge of SQL syntax
